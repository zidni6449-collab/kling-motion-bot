============================
KLING MOTION AI BOT
============================

CARA INSTALL & JALANKAN:

1. Install Python 3.10+
2. Buka terminal/CMD
3. Masuk ke folder ini
4. Jalankan:
   pip install -r requirements.txt
5. Jalankan bot:
   python bot.py

============================
PERINTAH ADMIN:
============================
/orders  - Lihat semua order
/kirim ORDER_ID  - Kirim video ke pembeli
   (harus reply ke video dulu, lalu ketik /kirim 5)

============================
CARA PAKAI SEHARI-HARI:
============================
1. Pembeli order lewat bot
2. Bot kirim notif + foto + audio ke kamu
3. Kamu proses di @BumiUnlimitedBot
4. Setelah video jadi, forward video ke bot kamu
5. Reply video itu dengan /kirim ORDER_ID
6. Video otomatis terkirim ke pembeli!

============================
HOSTING GRATIS DI RAILWAY:
============================
1. Daftar di railway.app
2. New Project > Deploy from GitHub
3. Upload file bot.py & requirements.txt
4. Set start command: python bot.py
5. Deploy!
